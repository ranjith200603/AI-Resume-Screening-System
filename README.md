# AI Resume Screening & ATS System

## Overview
This project automates resume screening using AI and workflow automation.

## Features
- Resume Upload
- PDF Data Extraction
- AI Resume Analysis
- Resume Scoring
- Candidate Classification
- Automated Emails
- Google Sheets Storage

## Tech Stack
- n8n
- OpenAI
- Gmail
- Google Sheets
